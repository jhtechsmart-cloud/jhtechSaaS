01시스템 소개

이 시스템은 (주)재현테크의 장비 견적부터 납품·수금, A/S와 소모품까지 한 곳에서 관리하는 업무 도구입니다.

이 시스템으로 할 수 있는 일

  • 고객이 홈페이지(공개 포털)에서 넣은 견적 요청·A/S·소모품 신청을 자동으로 접수받습니다.
  • 접수된 의뢰에 담당자를 배정하고, 견적서를 만들어 PDF로 발행하고, 고객에게 메일로 발송합니다.
  • 납품·수금 진행 상황을 상태로 추적하고, 고객 정보와 보유 장비를 관리합니다.
  • 장비 카탈로그·재고·소모품·데모예약을 관리합니다.

사용자 종류

구분할 수 있는 일
관리자모든 기능. 사용자 계정·권한 관리, 카탈로그·재고 편집, 견적 삭제까지 가능.
영업담당자본인이 맡았거나 아직 주인이 없는 의뢰를 처리. 견적 작성·메일 발송·고객 등록 가능. 계정 관리·카탈로그 편집·삭제는 불가.
ⓘ
이 설명서에서 관리자 표시가 붙은 기능은 관리자 권한이 있어야 보입니다. 영업담당자 화면에는 나타나지 않을 수 있어요. 권한이 더 필요하면 관리자에게 요청하세요.

02로그인과 내 계정

발급받은 이메일과 비밀번호로 로그인합니다.

로그인

  1. 브라우저에서 시스템 주소로 접속하면 로그인 화면이 나옵니다.
  2. 이메일과 비밀번호를 입력합니다.
  3. 아이디 저장에 체크하면 다음에 접속할 때 이메일이 자동으로 채워집니다.
  4. 로그인 버튼을 누르면 대시보드로 들어갑니다.
✓
비밀번호가 틀리면 "이메일 또는 비밀번호가 올바르지 않습니다" 안내가 뜹니다. 입력한 이메일은 그대로 남아 있으니 비밀번호만 다시 입력하면 됩니다.

최초 로그인 — 비밀번호 변경

관리자가 새 계정을 만들면 임시 비밀번호가 발급됩니다. 그 비밀번호로 처음 로그인하면 비밀번호 변경 화면이 먼저 뜹니다. 새 비밀번호(8자 이상)를 정하고 변경하고 시작하기를 누르면 시스템에 들어갑니다.

내 계정 메뉴

화면 오른쪽 위의 프로필 사진(또는 이니셜)을 누르면 계정 메뉴가 열립니다. 여기서 내 이름·권한을 확인하고, 계정 설정으로 이동하거나 로그아웃할 수 있습니다.

계정 설정에서 할 수 있는 것

  • 프로필 사진 등록 — 파일을 고르거나 끌어다 놓으면 됩니다(JPG·PNG 권장).
  • 비밀번호 변경 — 현재 비밀번호 + 새 비밀번호(8자 이상) + 확인을 입력합니다.

04대시보드 보는 법

로그인하면 가장 먼저 보이는 화면입니다. 오늘 무엇을 처리해야 하는지 빠르게 파악할 수 있습니다.

위쪽 요약 카드 4개

카드의미
처리 대기아직 손대지 않은 견적·A/S·소모품 신청 합계. 1건 이상이면 빨간색으로 강조됩니다. 누르면 견적 목록으로 이동.
진행 중 견적종료되지 않은 의뢰 건수와 견적 합계 금액.
이번 주 데모·납품이번 주 예정된 데모와 납품 건수.
전체 고객등록된 고객 수와 이번 달 새로 추가된 수.

2주 캘린더

앞으로 2주간의 일정을 색으로 보여줍니다. 위쪽 범례 버튼을 눌러 보고 싶은 종류만 켜고 끌 수 있습니다(설정은 다음 접속 때도 유지).

● 견적 ● A/S ● 소모품 ● 데모 ● 납품

그 밖의 위젯

  • 견적 파이프라인 — 단계별 견적 건수. "발송 후 7일이 지난 건"이 있으면 후속 연락 경고가 뜹니다.
  • 주간 활동 — 이번 주 신청을 요일별 막대로(1칸 = 1건).
  • 데모 및 납품 일정 / 이번 달 신청 / 최근 활동 — 클릭하면 해당 건 상세로 이동.
  • 미수금 — 계약했지만 아직 수금이 끝나지 않은 건 목록.
!
미수금 금액 안내: 아직 입금 관리 기능이 들어가기 전이라, 미수금 위젯의 금액은 정확한 미수금이 아니라 견적 합계를 보여줍니다(그래서 라벨도 "견적 합계"). 추후 수금 기능이 추가되면 실제 미수금으로 정확히 바뀝니다.

05의뢰(견적) 처리 흐름

"견적" 메뉴가 일상 업무의 중심입니다. 고객이 넣은 견적 요청이 여기로 들어오고, 접수부터 수금까지 8단계로 진행합니다.

의뢰 목록 화면

화면은 좌우로 나뉩니다. 왼쪽은 의뢰 목록, 오른쪽은 선택한 의뢰의 상세입니다.

  • 검색창: 업체명·접수번호·사업자번호로 찾기.
  • 탭: 진행중 / 완료 / 전체.
  • 목록은 오늘 / 이번 주 / 이전으로 묶여 표시됩니다. 아직 안 본 건에는 빨간 점이 붙습니다.
  • 행을 누르면 오른쪽에 상세가 열립니다. 아래로 더 보려면 더 보기.

8단계 진행 상태

접수→ 배정→ 견적중→ 견적발송→ 계약완료→ 수금중→ 수금완료
상태의미
접수고객이 견적을 요청해 막 들어온 상태. 담당자 배정 대기.
배정담당자가 정해진 상태. 견적 작성 대기.
견적중견적서를 작성 중(임시저장).
견적발송견적서를 발행해 고객에게 제시한 상태.
계약완료계약이 성사된 상태.
수금중수금을 진행 중.
수금완료모든 절차가 끝난 최종 완료.
종료중단·종결. 언제든 이 상태로 바꿀 수 있습니다.
ⓘ
견적서를 발행하면 상태가 자동으로 "견적발송"으로 바뀝니다. 그 이후(계약완료·수금중·수금완료)는 진행에 맞춰 직접 바꿔주세요.

담당자 배정과 상태 변경

의뢰 상세 위쪽 처리바에서 다룹니다.

  • 담당자 드롭다운으로 담당을 지정합니다. 주인이 없는 의뢰는 영업담당자가 "내가 맡기"로 가져올 수 있습니다.
  • 상태 드롭다운으로 단계를 바꿉니다.
!
담당자가 정해지지 않으면 상태를 바꿀 수 없습니다. "담당자를 먼저 배정해주세요" 안내가 보이면 담당부터 지정하세요.

의뢰 상세에서 보이는 정보

신청 기업 정보, 설치 환경 설문, 현장 사진, 선택한 장비와 옵션, 그리고 오른쪽 요약 패널(금액·견적 작성/수정·메일 발송·납품 일정·출고의뢰서 버튼)이 모여 있습니다.

✓
고객이 시스템에 등록돼 있지 않으면 "미등록 고객" 표시가 뜹니다. 고객으로 등록 버튼으로 거래처로 만들 수 있습니다(고객 등록 권한 필요).

06견적서 작성·발행

의뢰 상세 오른쪽 요약 패널의 견적 작성 버튼으로 시작합니다. 견적서 작성 권한 필요

작성 순서

  1. 견적 작성을 누르면 작성 화면이 열립니다. 고객이 요청한 장비가 기본으로 채워져 있습니다.
  2. 장비: 드롭다운에서 카탈로그 장비를 고르거나 "직접 입력"을 선택합니다. 단가는 자동으로 채워지고 수정할 수 있습니다. 수량을 입력하면 소계가 자동 계산됩니다. + 장비 추가로 줄을 더 넣을 수 있습니다.
  3. 포함 옵션: 기본으로 모두 체크돼 있습니다. 빼고 싶은 항목은 체크를 해제하면 견적서에서 제외됩니다.
  4. 추가 옵션: 별도 비용 항목(이름·단가·수량)을 직접 추가합니다.
  5. 견적서 사양 선택: 견적서 PDF에 넣을 사양 항목을 고릅니다. "한 페이지 예산"이 표시되며, 너무 많이 고르면 경고가 뜹니다.
  6. 오른쪽(또는 모바일 하단)의 합계를 확인하고 저장합니다.

임시저장 vs 발행하기

버튼결과
임시저장작성 중인 내용을 저장만 합니다(상태 변화 없음). 나중에 이어서 수정 가능.
발행하기견적서를 확정합니다. PDF가 만들어지고, 의뢰 상태가 "견적발송"으로 자동 변경됩니다.
ⓘ
발행하려면 장비가 최소 1줄 있어야 하고 금액이 올바라야 합니다. 빠진 항목이 있으면 안내 메시지가 표시됩니다.

견적 다시 만들기(재발행)

이미 발행한 견적을 고치려면 요약 패널의 수정을 누릅니다. 이전 내용이 그대로 채워진 채 열리고, 고쳐서 다시 발행하면 새 버전(v2, v3 …)으로 저장됩니다. 이전 버전도 그대로 남습니다.

07견적서 확인·메일 발송·버전

발행한 견적서는 PDF로 확인하고, 고객에게 메일로 보내고, 버전 이력을 볼 수 있습니다.

견적서(PDF) 확인

요약 패널의 견적서 확인 버튼을 누르면 PDF가 새 탭에서 열립니다. 누를 때마다 안전한 임시 링크가 새로 만들어집니다.

고객에게 메일 발송

메일 발송 권한 필요 · 발행된 견적서가 있어야 합니다.

  1. 요약 패널의 메일 발송을 누릅니다.
  2. 받는 사람·제목·본문을 확인합니다(고객 이메일과 제목은 자동으로 채워집니다). 필요하면 수정합니다.
  3. 발송을 누르면 견적서 PDF 링크가 담긴 메일이 담당자 명의로 나갑니다.
  4. 발송 상태가 "발송 중 → 발송됨"으로 표시됩니다. 주소를 바꿔 다시 보낼 수도 있습니다.
ⓘ
메일은 담당자의 하이웍스 계정으로 나가고 보낸편지함에도 남습니다. 담당자별 하이웍스 ID는 관리자가 사용자 관리에서 설정합니다(14장).

버전 이력

처리바의 버전 칩(예: "v2 (현재)")을 누르면 모든 버전과 발급일·금액·상태가 표로 보이고, 이전 버전과 무엇이 달라졌는지도 확인할 수 있습니다. 각 버전의 PDF도 따로 열 수 있습니다.

견적 삭제 관리자

버전이 하나면 견적 삭제, 여러 개면 이 버전 삭제 / 견적 전체 삭제 버튼이 있습니다. 삭제하면 PDF도 함께 지워지며 되돌릴 수 없습니다.

!
삭제는 되돌릴 수 없습니다. 확인 창에서 한 번 더 확인한 뒤 진행됩니다.

08출고의뢰서

납품 준비를 위한 출고의뢰서를 작성합니다. 출고의뢰서 작성 권한 필요

의뢰 상세 요약 패널의 출고의뢰서 버튼으로 엽니다(발행된 견적이 있어야 보입니다).

구성

  • 고객정보 — 회사명·장비·연락처·설치 일시·주소가 의뢰/견적에서 자동으로 채워집니다.
  • 장비 상세정보 — 프린터/커팅기에 따라 항목이 달라집니다(프린터: RIP·헤드·칼라·잉크 등 / 커팅기: 툴·카메라 등).
  • 기본 준비사항 — 운송차량·전기 사전준비·입고 준비물 등 체크.
  • 설치 현장정보 — 입고계획·출입문·전원·주차 등(설치 설문 내용이 초안으로 들어옵니다).

버튼결과

09고객 관리

"고객" 메뉴에서 거래처를 검색·등록·수정합니다. 고객 등록·수정 권한 필요

고객 목록

  • 통합 검색: 업체명·사업자번호·대표자·연락처·장부번호로 찾기(키보드 /로 검색창 포커스).
  • 필터: 지역, 담당영업, 빠른 필터(거래중/미배정/최근활동).
  • 컬럼(업체명·대표/담당·연락처·지역·최근활동)을 눌러 정렬할 수 있습니다.
  • 오른쪽 위 + 고객 등록으로 새 거래처를 추가, CSV 내보내기로 목록을 내려받습니다.

고객 상세

업체 기본 정보·연락처·주소와 함께, 오른쪽에 견적 / 보유장비 / A/S / 소모품 탭으로 그 고객의 모든 거래 이력을 봅니다. 위쪽 견적 작성 버튼으로 이 고객의 수기 견적을 바로 만들 수도 있습니다.

고객 등록·수정

  1. + 고객 등록(또는 상세의 수정)을 누릅니다.
  2. 기본 정보(업체명 필수)·연락처·사업장 정보를 입력합니다. 사업자번호와 전화번호는 입력하면 자동으로 형식이 맞춰집니다.
  3. 담당영업을 지정합니다.
  4. 보유장비를 표에 추가합니다(장비·시리얼번호·구입일자·설치주소).
  5. 아래 저장을 누릅니다. 저장하지 않고 나가려 하면 경고가 뜹니다.
✓
고객이 한 번도 없을 때는 "직접 입력" 또는 "견적요청에서 가져오기"로 시작할 수 있습니다. 보유장비를 잘 등록해 두면 고객이 A/S·소모품을 신청할 때 사업자번호만으로 장비가 자동으로 불러와집니다.

10A/S 신청 처리

고객이 홈페이지에서 넣은 A/S 신청을 접수·처리합니다.

  • 위쪽 카드로 전체 / 접수(미처리) / 진행중 / 완료 건수를 한눈에 보고, 카드를 눌러 바로 필터링합니다.
  • 업체명·접수번호·증상으로 검색하고, 상태·담당으로 거릅니다. 안 본 건에는 빨간 점이 붙습니다.
  • 행을 누르면 상세(고객 정보·증상·사진)가 열립니다.

처리 방법

  • 주인이 없는 신청은 맡기로 내 담당으로 가져옵니다.
  • 상태를 접수 → 진행중 → 완료로 바꿔가며 처리합니다.
ⓘ
"미확인" 표시가 붙은 신청은 고객 신원이 아직 검증되지 않은 건입니다. 연락(콜백)으로 확인하세요.

11소모품 신청 처리

고객이 보유 장비에 맞춰 신청한 소모품을 접수·처리합니다. A/S와 사용법이 같습니다.

  • 카드(전체/접수/진행중/완료), 검색(업체명·신청자·접수번호·품목), 상태·담당 필터.
  • 상세에서 업체 정보 + 신청 소모품(품목·수량) + 요청 메모를 확인합니다.
  • 주인 없는 건은 맡기, 상태는 접수 → 진행중 → 완료로 처리합니다.
ⓘ
소모품 단가·비용은 시스템에 표시되지 않습니다. 담당자가 확인 후 고객에게 별도로 안내합니다.

12데모예약 관리

데모센터(1곳) 예약을 달력으로 관리합니다. 같은 시간대에는 1건만, 15분 단위로 예약합니다.

  • 왼쪽 월간 달력에서 날짜를 누르면 오른쪽에 그날의 시간표(09:00~18:00)가 나옵니다. 데모는 보라색, 납품은 파란색 점으로 표시됩니다.
  • + 예약 등록으로 고객·장비·시간을 골라 새 예약을 만듭니다. 데모예약 권한 필요
  • 시간표의 예약 블록을 누르면 상세를 볼 수 있습니다.

13카탈로그 관리 관리자

견적·공개 홈페이지에 쓰이는 장비·소모품·분류·재고를 관리합니다.

장비

+ 새 장비로 등록합니다. 입력 항목: 장비명·모델·분류·기본가·상태, 사양(그룹별 항목, 견적서·홈페이지에 표시), 요약 불릿, 사진(첫 장이 대표 이미지), 견적서용 장비명·이미지, 제품 카탈로그 PDF, 옵션(포함/추가). 목록에서 행을 누르면 수정, 검색·상태 필터로 찾습니다.

재고현황

장비별 재고 수량·입고예정일·메모를 수기로 관리합니다(분류별로 묶여 표시). 수량이 0이면 자동으로 "품절"로 표시되고 입고예정일 칸이 강조됩니다. 행마다 따로 저장합니다.

ⓘ
영업담당자는 대시보드의 재고 링크로 재고를 읽기 전용으로 볼 수 있습니다(편집은 관리자만).

소모품

+ 새 소모품으로 등록합니다. 이름·단위·품번(SKU)·내부 참고가·상태·메모를 넣고, 적용 범위(어떤 분류 또는 장비에 쓰이는지)를 지정합니다. 적용 범위를 잘 지정해야 고객 소모품 신청에서 올바르게 매칭됩니다.

분류

장비 대분류(예: 프린터·커팅기) 아래 소분류를 둡니다. 대분류마다 견적 로고 종류(프린터/커팅기)를 정하면, 그 종류 장비의 견적서 좌상단 로고가 자동으로 맞춰집니다.

✓
프린터 견적에 프린터 로고가 찍히게 하려면 각 대분류에 "견적 로고"를 한 번 지정해 두세요.

14사용자·권한 관리 관리자

"사용자" 메뉴에서 직원 계정을 만들고 권한을 정합니다.

새 계정 만들기

  1. + 새 계정을 누릅니다.
  2. 이름·이메일(로그인 ID)·직책·연락처를 입력합니다.
  3. 권한을 고릅니다: 영업담당(추천 기본) / 관리자 / 직접설정.
  4. 계정 생성을 누르면 임시 비밀번호가 표시됩니다. 복사해서 본인에게 전달하세요(첫 로그인 때 본인이 바꿉니다).

  • 하이웍스 ID 설정 — 이 담당자가 견적 메일을 보낼 때 쓰는 하이웍스 계정.
  • 권한 변경, 계정 비활성화(본인 계정은 비활성화 불가).

15고객이 보는 화면(공개 포털)

고객은 로그인 없이 홈페이지(공개 포털)에서 신청합니다. 여기서 들어온 신청이 위에서 설명한 의뢰·A/S·소모품 목록으로 자동 접수됩니다. (참고용)

견적 요청

장비를 고르고 회사·사업자번호·연락처와 (선택) 설치 환경·현장 사진을 입력해 제출 → 견적 목록 "접수"로 들어옴.

A/S 신청

사업자번호로 보유 장비를 불러와 증상·사진을 넣어 제출 → A/S 목록으로.

소모품 신청

사업자번호 조회 후 보유 장비에 맞는 소모품을 골라 제출 → 소모품 목록으로.

장비 카탈로그

공개된 장비의 사진·사양·영상을 보고 바로 견적 요청으로 이동.

제출이 끝나면 고객은 접수번호(예: 견적 REQ-…, A/S AS-…, 소모품 SUP-…)를 받습니다.

공개 포털의 대표 연락처는 전화 02-839-7723, 이메일 support@jhtech.co.kr입니다.

16자주 묻는 질문 · 용어

자주 묻는 질문

Q. 비밀번호를 잊어버렸어요.

관리자에게 요청하면 사용자 관리에서 비밀번호 재설정으로 새 임시 비밀번호를 발급해 줍니다. 그 비밀번호로 로그인하면 새 비밀번호를 정하게 됩니다.

Q. 어떤 메뉴/버튼이 안 보여요.

권한이 없어서일 수 있습니다(예: 카탈로그·사용자 메뉴는 관리자만). 관리자에게 권한을 요청하세요.

Q. 상태를 바꿀 수 없어요.

담당자가 배정돼 있어야 상태를 바꿀 수 있습니다. 담당자부터 지정하세요.

Q. 견적서를 고치고 싶어요.

의뢰 상세의 수정 버튼으로 다시 작성해 발행하면 새 버전으로 저장됩니다. 이전 버전도 남습니다.

Q. 미수금 금액이 이상해요.

아직 입금(수금) 관리 기능 전이라 현재는 "견적 합계"를 보여줍니다. 추후 정확한 미수금으로 바뀝니다.
