# 하이웍스 메일 발송 연동 — 미팅 질문 시트 (v2, API 문서 확인 후)

**미팅 목적 한 줄** — 발행된 견적서 PDF를 **영업담당자 명의로 고객에게 메일 발송**하고, 그 메일이 **담당자 하이웍스 "보낸편지함"에 남게** 하는 기능을 만듭니다. API 문서를 확인해 대부분 풀렸고, **남은 건 아래 2가지뿐**입니다.

- 미팅 날짜: 2026-06-17
- 연동 대상: 가비아 하이웍스 메일 REST API (`POST /office/v2/webmail/sendMail`)
- 인증 방식: **오피스 토큰(Office Token)** 으로 확정

## 🔴 미팅에서 받아야 할 것 — 딱 2개

### A. 메일 발송 시 PDF "첨부" 가능한가요?
`sendMail` API 문서 바디에 `to / user_id / cc / bcc / subject / content / save_sent_mail` 만 있고 **첨부(파일) 필드가 없습니다.**

- 물을 것:
  1. body가 form-data인데, **파일 필드를 추가로 넣으면 첨부가 되나요?** (필드명·형식·용량 제한)
  2. 안 되면, 별도 **첨부 업로드 API**가 있나요?
  3. 둘 다 없으면 **본문(content)에 PDF 다운로드 링크**를 넣는 방식으로 가야 하나요?

### B. 오피스 토큰 발급 + 우리 서버 IP 등록 (관리자 작업)

> - `162.220.232.251`

## ✅ 이미 문서로 확인된 것 (미팅에서 안 물어도 됨)

| 항목 | 확정 내용 |
| --- | --- |
| 인증 방식 | **Office Token 하나.** `Authorization: Bearer {office_token}`. OAuth(Access Token)는 deprecated·메일 미지원 |
| 엔드포인트 | `POST https://api.hiworks.com/office/v2/webmail/sendMail`, body = **form-data** |
| 개인 명의 발송 | `user_id`(보내는 사람 하이웍스 계정 ID) 파라미터로 담당자 지정 |
| 보낸편지함 저장 | `save_sent_mail=Y` (Scope 설명에도 "보낸 편지함 보관 가능" 명시) |
| 성공 판정 | 응답 `{ "code": "SUC", "result": { "successList": [...] } }` |
| 발송 한도 | **오피스당 1,000건/일**(성공 기준) — 견적 발송엔 충분 |
| 본문 | `content` 필수(String). HTML 지원 여부만 가능하면 같이 확인 |
